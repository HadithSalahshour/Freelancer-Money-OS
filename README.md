Freelancer Money OS
A fully local, privacy-first financial dashboard built specifically for freelancers. No accounts, no cloud, no third-party access. Just open the file and start managing your money.
The Problem
Freelancers deal with irregular income, self-managed taxes, unpaid invoices, and no financial safety net. Most budgeting tools are built for salaried employees and don't address the reality of freelance finances.
Features

Real take-home income tracking after taxes and fees
Automatic tax set-aside calculations
Runway tracker to know how long your cash lasts
Invoice management with outstanding balance tracking
Subscription tracking with renewal alerts
Income smoother chart with 3-month average
Goals and buffers including emergency fund and slow-month buffer
Action panel that surfaces what needs attention
Multiple clients, income streams, and bank accounts
Full data export and import backup

Built With
HTML, CSS, JavaScript
